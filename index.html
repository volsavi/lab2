Представлення бакалаврської роботи

Цей сайт створений для презентації результатів бакалаврської роботи студента спеціальності "Комп’ютерні науки". Робота присвячена дослідженню ефективності вебтехнологій у розробці односторінкових застосунків.

Актуальність теми

У сучасному цифровому суспільстві веб-застосунки є основним інструментом взаємодії з користувачами. Вивчення ефективності односторінкових рішень є важливим з точки зору продуктивності, UX та доступності.

Мета та завдання

Мета: Розробити функціональний, адаптивний та доступний веб-сайт для представлення бакалаврської роботи.

  • Ознайомитися з принципами сучасної веб-розробки
  • Реалізувати GitLab Flow для командної роботи
  • Оптимізувати сайт для SEO та доступності
  • Забезпечити багатомовність

Методологія дослідження

У процесі роботи було використано такі підходи:

  • Аналіз кращих практик веб-розробки
  • Використання системи контролю версій Git
  • Застосування принципів семантичної розмітки та адаптивного дизайну
  • Валідація за стандартами W3C та Lighthouse

Очікувані результати

Фінальний результат — повністю задокументований сайт, який демонструє навички розробки, вміння працювати з git та дотримання сучасних стандартів вебу. Ресурс буде зручний для перегляду на різних пристроях та легкий для індексації пошуковими системами.
